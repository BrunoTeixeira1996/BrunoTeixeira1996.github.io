Yelp Restaurant Searcher

I had an idea about making a script with python that could be usefull to me. I was reading documentation from zomato API and i decided to give it a try. I spent a couple of hours building that project but i wasnt happy at all. The api wasn't bad but had alot of restrictions(1000 calls/day). I googled "Apps like zomato" and i found Yelp. I readed the API documentation and i put my hands to the business. From this project i learned some cool new stuff (how should i use sys.argv to user input, how to work with the json response, how to work with requests). Like always the most important thing is to pratice and keep learning new things, because those things will be important in the end. Like always the source code is on my github.
