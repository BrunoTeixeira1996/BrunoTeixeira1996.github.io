Active Directory Searcher


One of my collegues at work built a script in batch with psexec to search for users attributes in Active Directory. That tool is realy helpful because we need to see whats happening on an account when someone calls us telling that cant login in their windows account. So, i saw the batch script and i was thinking if i could build something better with python. Finally i created the Active Directory Searcher based on that batch script. Learned alot while building the script and it works like a charm at work. Github repository
