A Bot to Discord

A Bot made with python to do some data analysis on a discord server. The main purpose is to analyse the time that most people are online on the server. I built some basic commands as it shows in the repository. There is a cronjob that tracks every 15 seconds how many people are online, iddle and offline in the server. The final result is supercool, since i was able to work with new libraries that i never used such as pandas and matplotlib. To use this bot you need to change some values and you need to create a new bot in discord API.
